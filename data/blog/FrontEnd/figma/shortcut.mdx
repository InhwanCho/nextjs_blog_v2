Figma는 디자이너들이 선호하는 UI/UX 디자인 툴 중 하나입니다. Mac 사용자를 위한 Figma 단축키를 알고 있으면, 디자인 작업의 속도와 효율성을 크게 향상시킬 수 있습니다.

이 글에서는 Figma 사용 시 유용한 몇 가지 주요 단축키를 소개하고자 합니다.

## 단축키 비교 표

| 기능                    | 단축키                       |
| ----------------------- | ---------------------------- |
| 복사 & 붙여넣기         | `Cmd` + `C`, `Cmd` + `V`     |
| 복사 & 붙여넣기(동일)   | `Option`+ 드레그             |
| 인라인 레이어 선택      | `Cmd` + 클릭                 |
| 여러 레이어 선택        | `Shift` + 클릭               |
| 모든 레이어 선택        | `Cmd` + `A`                  |
| 레이어 순위 변경        | `Cmd` + `]` or `[`           |
| 레이어 순위 변경(동일)  | `Cmd` + `Option` + `위 아래` |
| 실제 크기 보기          | `Cmd` + `1`                  |
| 전체 화면 모드로 전환   | `Cmd` + `Shift` + `.`        |
| 그룹화                  | `Cmd` + `G`                  |
| 그룹화 해제             | `Cmd` + `Backspace`          |
| 수평 정렬               | `Cmd` + `Shift` + `H`        |
| 수직 정렬               | `Cmd` + `Shift` + `V`        |
| 벡터 도구               | `V`                          |
| 프레임 도구             | `F`                          |
| 슬라이스 도구           | `S`                          |
| 텍스트 도구             | `T`                          |
| 이미지 추가             | `Cmd` + `Shift` + `K`        |
| 아이 디자인(스포이드)   | `I`                          |
| 줌 인                   | `Cmd` + `+`                  |
| 줌 아웃                 | `Cmd` + `-`                  |
| 컴포넌트로 만들기         | `Cmd` + `Option` + `K`          |
| 사용된 컴포넌트 찾기    | `Cmd` + `Shift` + `F`        |
| 검색                    | `Cmd` + `P`                  |
| 클릭된 요소 범위 줄이기 | 더블 클릭                    |

### 버튼 만들기

버튼을 만들때는 rectangle과 text를 이용하여 만들어도 되지만, 일반적으로 `frame`과 `text`의 조합으로 생성하는것을 선호합니다.

생성 후, `auto layout`의 `+ 버튼`을 눌러서 알맞은 크기로 자동으로 생성합니다.

### 컴포넌트

메인 컴포넌트를 변경하면 인스턴스는 자동으로 변경된다. 단, 반대의 로직은 불가능합니다.

## Relume을 활용한 디자인 초안 생성

`Relume`은 디자인 시스템을 더욱 효율적으로 관리 및 생성할 수 있도록 돕는 AI 사이트입니다.

`Relume`에서 sitemap 및 디자인 초안을 AI로 생성 후 figma로 export하면 보다 쉽게 초안을 생성할 수 있습니다.
